A powerful Lead Management Dashboard addon for Forminator. Track SEO leads, manage feedback, and categorize leads as positive/negative.

== Description ==

**Forminator Lead Dashboard** is an open-source addon plugin for Forminator that adds a complete lead management system to your WordPress site.

### Features

* **Dashboard Overview** - See all your lead statistics at a glance
* **Lead Tracking** - Track all Forminator form submissions as leads
* **Status Management** - Categorize leads as New, Positive, Negative, Follow Up, Converted, or Closed
* **Sales Team Feedback** - Your team can add feedback with ratings (👍 Positive, 😐 Neutral, 👎 Negative)
* **Activity Logging** - Track all changes made to leads
* **Filtering & Search** - Filter by form, status, date range, and search
* **Bulk Actions** - Update multiple lead statuses at once
* **CSV Export** - Export leads to CSV for reporting
* **Charts & Analytics** - Visual charts showing lead trends and status distribution

### Requirements

* WordPress 5.0+
* PHP 7.4+
* Forminator plugin (free version works)

== Installation ==

1. Upload the `forminator-lead-dashboard` folder to `/wp-content/plugins/`
2. Make sure Forminator is installed and activated
3. Activate the "Forminator Lead Dashboard" plugin
4. Go to "Lead Dashboard" in your admin menu

== Usage ==

### Dashboard
The main dashboard shows:
- Total leads, new leads, positive/negative counts
- Conversion rate
- Leads over time chart
- Leads by status pie chart
- Top forms by lead count
- Recent leads table

### Managing Leads
1. Go to Lead Dashboard > All Leads
2. Use filters to find specific leads
3. Click "View" to see lead details
4. Update the lead status
5. Add feedback from your sales team

### Adding Feedback
1. Open a lead's details
2. Select a rating (Positive, Neutral, or Negative)
3. Write your feedback
4. Click "Add Feedback"

### Lead Statuses
- **New** - Newly submitted leads
- **Positive** - Qualified, interested leads
- **Negative** - Unqualified or not interested
- **Follow Up** - Needs follow-up action
- **Converted** - Successfully converted to customer
- **Closed** - Lead archived/closed

== Frequently Asked Questions ==

= Does this work with Forminator free version? =
Yes! This plugin works with both free and pro versions of Forminator.

= Can multiple team members add feedback? =
Yes, any user with admin access can add feedback to leads.

= How do I export my leads? =
Go to All Leads, apply any filters you want, then click "Export CSV".
